Untuk menjalankan aplikasi secara lokal, ikuti langkah-langkah berikut:

1. **Clone repositori ini**

   ```bash

   cd nongki-geh
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Jalankan aplikasi**

   ```bash
   npm start
   ```

   Aplikasi akan berjalan di `http://localhost:3000`.
